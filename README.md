# El-Centavo
¿Desde cuándo existe el centavo en Estados Unidos?  El centavo fue la primera moneda oficial emitida por Estados Unidos.  Año de inicio: 1793  Fue creado tras la Ley de la Casa de la Moneda de 1792, que estableció el sistema monetario estadounidense.  Desde el principio representó 1/100 de dólar, función que mantiene hasta hoy.
